# Github Battle!!!

## Install

    $ git clone
    $ cd PROJECT
    $ npm install

## Start & watch

    $ npm start

## Simple deployment to firebase for production

    $ npm deploy

## Languages & tools

- [React](http://facebook.github.io/react) is used for UI.
- [Webpack](https://webpack.js.org/)
- [Babel](https://babeljs.io/)
- [Axios](https://www.npmjs.com/package/axios)
